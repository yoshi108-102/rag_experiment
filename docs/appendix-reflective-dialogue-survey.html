APPENDIX A / LITERATURE SURVEY

Reflective Dialogue Prompt 関連研究サーベイ

対象: 学習者の振り返り支援に向けた prompt 設計。特に OARS、reflective listening、 autonomy support、psychological safety、trauma-informed / safety-aware、 および LLM / chatbot 実装の知見を対象に整理。
検索日: 2026-03-09。一次ソースを優先し、PubMed / PMC / DOI 公開ページを中心に手動選定。

1. サーベイ範囲

選定基準

  • 対話設計の原理に直接つながる研究を優先
  • レビュー、メタ分析、ランダム化比較、実装研究を混在させて確認
  • 教育現場、医療対話、行動変容支援、debriefing を含めて横断的に選定

今回の問い

  • 受容と反射を prompt に明示する根拠はあるか
  • 質問数を絞り、自己決定感を守る方針に根拠はあるか
  • safety-aware / trauma-informed の制約はどこまで強く書けるか
  • LLM に reflective responses を任せるとき何がボトルネックか

2. 要点

比較的強い支持

MI 系の研究では、効果の中核は「共感的な関係性」と「本人の change talk を引き出す技術」に置かれている。

Inference: prompt では解決策提示より先に、受容と反射を優先する設計が妥当。

比較的強い支持

教育領域のメタ分析では、teacher autonomy support は学生の need satisfaction と self-determined motivation と強く結び付く。

Inference: 「話す範囲を相手が選べる」設計は保持すべき。

中程度の支持

psychological safety 研究は、学習志向、支援、包摂性、階層の弱化が発話しやすさを支えると示す。

Inference: 詰問調や能力不足を示唆する文言は避けるべき。

運用上かなり有用

PEARLS は、self-assessment と focused discussion を土台にした debriefing の実装枠組みを提供する。

Inference: 質問は1つずつ、場面を限定しすぎず、自己評価を促す順番が合う。

根拠はあるが比較研究は薄い

trauma-informed teaching では再トラウマ化回避の必要性は強く示される一方、どの技法が最も有効かの比較証拠はまだ薄い。

Inference: safety ルールは強く書きつつ、細かな戦術は断定しすぎない方がよい。

実装上の重要な注意

LLM に reflective responses を作らせること自体は有望だが、品質チェッカーなしでは不適切な反射が混ざる。

Inference: prompt だけでなく guard / validator が必要。

MIの理論 Miller & Rose (2009)
PubMed / PMC 		理論レビュー MI の有効成分を「relational component focused on empathy」と「change talk の喚起」に整理している。 受容と反射を前段に置き、ユーザー自身の言語化を促す質問に寄せる。 High
MIの効果 Rubak et al. (2005)
PubMed / PMC 		系統的レビュー + メタ分析 72件の RCT を統合し、MI はおよそ 4 件中 3 件で臨床的効果を示し、15分程度の短時間介入でも一定の効果があると報告。 短い1ターンでも、受容的な働きかけ自体に意味があると見なせる。 High
自己決定感 Bureau et al. (2022)
PubMed / PMC 		メタ分析 144研究、79,000人超を対象に、teacher autonomy support は学生の need satisfaction と self-determined motivation を強く予測すると報告。 「どこまで話すかを相手が選べる」「無理に深掘りしない」を prompt の中心に置く。 High
心理的安全性 O'Donovan & McAuliffe (2020)
PubMed / DOI 		系統的レビュー 36論文から 13 の enabler を整理し、learning orientation、support、hierarchy / inclusiveness などを心理的安全性の条件として抽出。 評価・詰問・断定を抑え、「間違っていても話せる」空気を文面で作る。 High
振り返り支援 Eppich & Cheng (2015)
PubMed / DOI 		実装フレームワーク PEARLS は debriefing を learner self-assessment、focused discussion、directive feedback / teaching に分けて構造化している。 現行 prompt は self-assessment と focused discussion を主軸にし、teaching は原則オフでよい。 Medium
trauma-informed Gladden et al. (2023)
PubMed / DOI 		系統的レビュー 27論文をレビューし、self-care や journaling などの実践が提案される一方、個別手法の有効性比較は乏しいと指摘。 安全配慮の原則は強く書くが、特定技法を万能視しない。過剰な深掘り回避を優先する。 Medium
trauma-informed Henshaw (2022)
PubMed / PMC 		レビュー / 実践提案 higher education で culturally responsive な trauma-informed approaches を適用するための概念整理を行っている。 「安全」と「包摂性」を分けずに扱い、相手の語彙保持や決めつけ回避を明記する。 Medium
chatbot実装 Almusharraf et al. (2020)
PubMed / PMC 		反復設計研究 反射文と summary statement を含む MI chatbot を反復改良し、分類 F1 を 0.63 から 0.82 に改善した。 会話ログから改善する前提は妥当。prompt だけでなく失敗例の収集が重要。 Medium
chatbot実装 Brown et al. (2023)
PubMed / PMC 		反復実装研究 generative reflections を入れたバージョンは、question-only バージョンより perceived empathy が高く、extended version では readiness と importance も上がった。 反射的要約は飾りではなく、質問だけより価値がある可能性が高い。 Medium
LLM reflection Kumar et al. (2024)
PubMed / PMC 		アルゴリズム開発 + 検証 GPT-4 に prompt engineering で backward-looking complex reflections を生成させ、150件中 132件が acceptable。残り 12% は不適切で checker が必要と結論。 現行 system prompt の改善だけでなく、multi-question、過解釈、押しつけを弾く後段 guard が必要。 High
LLM / chatbotの限界 He et al. (2022)
PubMed / PMC 		ランダム化比較試験 MI-style chatbot は neutral chatbot に対して engagement / alliance / empathy で有意差が出なかった一方、会話自体で motivation は上がり、communication competence は主要指標と正に関連。 MI 風の表面文体だけでは足りず、応答の妥当性と一貫性が重要。prompt は style と quality control をセットで考える。 Medium

4. 現行promptへの示唆

残すべき方針

  • 受容してから進む
  • 質問は1つだけにする
  • 「なぜ」より「いつ / どこで / 何が見えていたか」を先に聞く
  • 本人の表現を保持して返す
  • 終了意図が見えたら `FINISH` を優先する

追加で必要な運用

  • 過解釈、原因断定、複数質問を弾く validator
  • frustration / closure の rule-based override
  • good / bad response のログ収集と継続的な微修正
  • 将来的には reflection quality の rubric を導入
ここでの「転用の確からしさ」は、このリポジトリの用途に引き付けた実務上の推定です。文献そのものが 直接に「学習者向け reflective prompt」を比較したわけではない点は残ります。

5. 限界

  • MI の主要エビデンスは医療・行動変容文脈で、学習振り返りそのものへの直接転用ではない。
  • trauma-informed は必要性の論拠は強いが、具体的な会話マイクロスキルの比較研究はまだ少ない。
  • chatbot 研究の多くは smoking cessation に偏っており、教育現場で同じ効果量が出るとは限らない。
  • LLM reflection 研究は有望だが、品質検証なしで運用できる水準とはまだ言いにくい。
